collective.polls Installation
-----------------------------

To install collective.polls using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.polls`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.polls
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
